E[X] Showdown is Northeastern's first algorithmic poker tournament. Write a Python bot that plays No-Limit Texas Hold'em. No manual play — your code makes every decision. Compete against other students' algorithms in automated matches. Build your bot. Beat everyone else's.

// Why Compete?

Game Theory in Practice

Poker is incomplete information. Your bot has to make decisions under uncertainty — the same skill that drives prediction markets, trading, and research.

Build Something Real

Not a class project. Not a tutorial. A bot that competes head-to-head against other students' code. Ship it, test it, watch it play.

All Majors Welcome

CS, finance, math, polisci, philosophy — good judgment doesn't require a specific major. Some of the best bots come from people who've never played a hand.

Sharpen Your Edge

The firms that recruit from competitions like this are the same ones hiring for quant, SWE, and research roles. This is how you stand out when everyone has the same GPA.

// How It Works

01

Submit Your Bot

Write a decide() function in Python. It receives the game state, returns an action. That's the entire interface. Submit a single bot.py file.

02

Qualify

All bots play in a Swiss-system tournament. Multiple rounds, 400 hands per match, 6-bot tables. Bots are ranked by cumulative chip delta. Top performers advance.

03

Finals

The best bots compete in a bracket. Best algorithm wins. Results are streamed live so everyone can watch the action.
