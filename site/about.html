// Info

E[X] Showdown is an algorithmic poker tournament run by E[X], Northeastern's prediction markets club. You write a Python bot that plays No-Limit Texas Hold'em. The engine handles everything else — dealing, betting, hand evaluation, tournament pairing, and scoring.

Format

GameNo-Limit Texas Hold'em (6-max tables)
LanguagePython 3.10
InterfaceOne function: decide(game_state) → action
QualifierSwiss-system tournament, 400 hands per match
RankingCumulative chip delta across all rounds
FinalsSingle-elimination bracket, streamed live
EligibilityNortheastern students, all majors

Your Bot

You submit one file — bot.py — with one function:

def decide(game_state: dict) -> dict:
    # your entire strategy goes here
    return {"action": "call"}

The engine calls decide() once per action. You get the full game state — your cards, community cards, pot size, stack sizes, betting history, position. You return one action.

Game State

your_cardsYour two hole cards, e.g. ["As", "Kh"]
community_cardsBoard cards, e.g. ["7d", "Tc", "2s"]
streetpreflop / flop / turn / river
potTotal chips in the pot
your_stackYour remaining chips
amount_owedChips needed to call (0 = free check)
can_checkTrue when no bet to call
min_raise_toMinimum legal raise total
playersPublic info on all seats
action_logEvery action taken this hand

Valid Actions

{"action": "fold"}
{"action": "check"}                  # only when can_check is True
{"action": "call"}
{"action": "raise", "amount": 1200}  # total bet, not raise-by
{"action": "all_in"}

Invalid or missing actions default to fold. Raises below the minimum are snapped up automatically.

Rules

  • 2 seconds to return an action or your bot auto-folds
  • No network calls during gameplay
  • No file writes — filesystem is read-only at runtime
  • 768 MB RAM, 0.5 CPU core per bot
  • Crashes and exceptions auto-fold for that hand — your bot stays in the tournament

Available Libraries

eval7 · numpy · scipy · treys · scikit-learn

Not allowed: No external API calls (no LLMs, no webhooks). No reading other bots' code or cards. No file writes. No threading tricks to dodge the timeout. No collusion between bots. Violations = disqualification.

Submission

Submit either:

  • bot.py — single file, simple bot
  • bot.zip containing bot.py + optional data/ — for bots shipping lookup tables, neural net weights, CFR blueprints, etc.

data/ constraints: 200 MB max, no .py files, read-only at runtime, loaded at import time only via os.environ["BOT_DATA_DIR"].

Sample Submission — demo.py

Here's the full demo.py that ships with the engine repo. It spins up a local Flask server, runs matches between built-in bots, and shows results live. Study it to understand how the engine calls your bot and how matches work end-to-end.

""" E[X] Showdown — Local Demo Run: python demo.py Open: http://localhost:5001 No Docker, Redis, or Supabase needed. Runs real matches using the actual game engine and shows results live. """ import sys, os, json, time, threading, uuid, random sys.path.insert(0, os.path.dirname(__file__)) from flask import Flask, Response, jsonify, render_template_string from sandbox.match import run_match from engine.tournament import swiss_pairing, compute_standings, select_finalists app = Flask(__name__) # --------------------------------------------------------------------------- # State (in-memory for demo) # --------------------------------------------------------------------------- BOT_PATHS = { "The Aggressor": "bots/aggressor/bot.py", "The Mathematician":"bots/mathematician/bot.py", "The Shark": "bots/shark/bot.py", "Template Bot A": "bots/template/bot.py", "Pot-Odds Bot B": "bots/ref_bot_2/bot.py", "Template Bot C": "bots/template/bot.py", } state = { "log": [], # event log (SSE stream) "standings": [], # current leaderboard "hands": [], # last match hand history "running": False, "round": 0, } log_lock = threading.Lock() def emit(msg, kind="info"): with log_lock: state["log"].append({"t": time.time(), "msg": msg, "kind": kind}) # --------------------------------------------------------------------------- # Routes # --------------------------------------------------------------------------- @app.route("/") def index(): return render_template_string(PAGE) @app.route("/stream") def stream(): """SSE endpoint — pushes log events to the browser in real time.""" def generate(): last = 0 while True: with log_lock: new = state["log"][last:] last = len(state["log"]) for entry in new: yield f"data: {json.dumps(entry)}\n\n" time.sleep(0.2) return Response(generate(), mimetype="text/event-stream", headers={"Cache-Control": "no-cache"}) @app.route("/run/match", methods=["POST"]) def run_single_match(): """Run one 6-player match and return updated standings.""" bot_ids = list(BOT_PATHS.keys()) match_id = f"demo_{uuid.uuid4().hex[:8]}" emit(f"Starting match {match_id}...", "dim") t0 = time.time() result = run_match(match_id, BOT_PATHS, n_hands=150, verbose=False) elapsed = time.time() - t0 emit(f"Match complete in {elapsed:.1f}s", "dim") # Update standings all_results = [] for bid, delta in result["chip_delta"].items(): all_results.append({"bot_id": bid, "bot_path": BOT_PATHS.get(bid,""), "chip_delta": delta}) prev = {s["bot_id"]: s for s in state["standings"]} new_standings = compute_standings( [{**s, "chip_delta": s["cumulative_delta"]} for s in state["standings"]] + all_results ) state["standings"] = new_standings # Log results sorted_res = sorted(result["chip_delta"].items(), key=lambda x: -x[1]) for bid, delta in sorted_res: sign = "+" if delta >= 0 else "" kind = "win" if delta > 0 else "err" if delta < 0 else "dim" emit(f" {bid:22s} {sign}{delta:,}", kind) state["hands"] = result.get("hands", []) return jsonify({ "standings": new_standings, "hands": result.get("hands", []), }) @app.route("/run/tournament", methods=["POST"]) def run_tournament(): """Run a 3-round Swiss tournament across all bots.""" bot_list = [{"bot_id": bid, "bot_path": path, "cumulative_delta": 0, "matches_played": 0} for bid, path in BOT_PATHS.items()] state["standings"] = [] all_results = [] for rnd in range(1, 4): state["round"] = rnd emit(f"=== ROUND {rnd} ===", "bold") standings_for_pairing = compute_standings(all_results) if all_results else bot_list tables = swiss_pairing(standings_for_pairing, table_size=min(6, len(bot_list))) emit(f" {len(tables)} table(s) this round", "dim") for t_idx, table in enumerate(tables): bot_paths_for_match = {b["bot_id"]: b["bot_path"] for b in table} match_id = f"t_r{rnd}_t{t_idx}" emit(f" Table {t_idx+1}: {', '.join(bot_paths_for_match.keys())}", "dim") result = run_match(match_id, bot_paths_for_match, n_hands=150) for bid, delta in result["chip_delta"].items(): all_results.append({ "bot_id": bid, "bot_path": BOT_PATHS.get(bid, ""), "chip_delta": delta, }) sign = "+" if delta >= 0 else "" kind = "win" if delta > 0 else "err" if delta < 0 else "dim" emit(f" {bid:22s} {sign}{delta:,}", kind) final_standings = compute_standings(all_results) state["standings"] = final_standings finalists = select_finalists(final_standings, n=3) emit("=== FINALISTS ===", "bold") for i, f in enumerate(finalists): emit(f" #{i+1} {f['bot_id']} ({f['cumulative_delta']:+,})", "win") return jsonify({ "standings": final_standings, "round": 3, "finalists": len(finalists), "hands": state.get("hands", []), }) # --------------------------------------------------------------------------- # Entry point # --------------------------------------------------------------------------- if __name__ == "__main__": print("\n" + "="*50) print(" E[X] SHOWDOWN — LOCAL DEMO") print("="*50) port = int(os.environ.get("DEMO_PORT", "5001")) print(f" Open: http://localhost:{port}") print(" Bots: ", ", ".join(BOT_PATHS.keys())) print("="*50 + "\n") app.run(debug=False, threaded=True, port=port)

To run it locally: python demo.py then open http://localhost:5001. You'll see a terminal-style UI where you can run single matches or a full 3-round Swiss tournament between the built-in bots.
